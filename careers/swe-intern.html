Savour! Software Engineering Internship Listing

Job function: Web and App Development 

Brief job description: As a software engineer intern at Savour!, you will play an active role in bringing value-added technical solutions. You will gain first-hand experience and seek innovative ways to include automation in processes while optimizing the platform and enhancing the user experience. You will have the opportunity to work alongside the CTO directly to gain valuable insights on how to build a scalable and high-performance software platform and have opportunities to suggest and implement new initiatives.

Roles and responsibilities: 

  1. Work with React.js for web front end
  2. Work with react native for mobile front end
  3. Work closely with a small team and participate in building scalable back-end systems using django for mobile/web applications
  4. Work with web servers on AWS
  5. Work with REST APIs
  6. Ensure excellent code quality
  7. Assist and support the team with testing, troubleshooting and optimizing the apps to make sure they run smoothly on mobile/web platform
  8. Assist and support the team with DevOps activities
  9. Rapidly build, iterate, test, and deploy new features and products in the front-end and back-end of the platform.
  10. Work closely with the founders and participate in planning our technical strategy, roadmap, and timeline for the next phase

Learning Objectives & Outcomes

  1. Professional Development
  2. Gain insights into the local market, B2B e-commerce industry and social entrepreneurship
  3. Enhance coding skills; get exposed to new tools and programming languages
  4. Hands-on experience to apply computer science studies in a professional environment
  5. Improve presentation and communication (both oral and written) skills
  6. Gain experience in developing applications via being fully involved in both frontend and backend development
  7. Learn how to translate user requirements into the overall architecture and implementation of Savour!'s platform
  8. Hone critical thinking skills by ideating feasible solutions to further improve the design and functions of current systems

Requirements:

  • Students who are currently pursuing a Bachelor’s degree or Polytechnic Diploma in Computer Science or a related technical field
  • Has a grasp of web technologies, react.js is a must
  • Knowledge of python is a must 
  • Good communication skills, problem-solving and creative thinking abilities, and the ability to respond to request in a short time
  • Students from non-computing engineering faculties are welcome if they are comfortable with react.js
  • Knowledge of react native (preferred)
  • Prior internships / hack-a-thons / freelancing / own project experience (preferred)
  • Experience with Github and APIs (preferred)
